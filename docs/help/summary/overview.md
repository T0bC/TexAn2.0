# Summary Statistics

Summary statistics for each measurement column, grouped by descriptive columns.

## How to Use

1. **Group by** — Select one or more descriptive columns to group the statistics
2. **Test for Normality** — Enable the Shapiro-Wilk test to check if measurements follow a normal distribution (p-value < 0.05 indicates non-normal)
3. Results appear as one card per measurement column

## Statistics Computed

- **n** — Number of observations (after filtering)
- **mean** — Arithmetic mean
- **median** — Median value
- **var** — Variance
- **sd** — Standard deviation
- **sem** — Standard error of the mean
- **cv** — Coefficient of variation (sd / mean)

## Downloads

- Use the download icon on each card to export a single table as XLSX
- Use **Download All Tables** in the sidebar to export all tables into a single XLSX workbook (one sheet per measurement)
