1. Upload your file using the `Browse` button.
2. For `CSV` files, ensure to select the appropriate delimiter.
3. For `Excel` files, the **first sheet is assumed to be the data** sheet.
4. Please review your data frame before proceeding.

#### Note

The app identifies descriptive columns (non-measured values or group-assignable data) as those written in CAPITAL LETTERS. Underscores are acceptable, but avoid special characters like \# in column names. Measurement columns should have mixed case names (both upper and lower case letters), with underscores permitted.
