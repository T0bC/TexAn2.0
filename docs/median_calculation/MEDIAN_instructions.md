Grouping defines how your data is organized before calculating summaries like medians. It reflects the structure of your research design — for example, samples may differ by material, treatment, sex, species, or location.

* Select grouping columns that uniquely identify each sample or condition.
* For more granular analysis, include columns like bone type or tooth position.
* For broader summaries, you can exclude finer details and group only by high-level categories.

*Example*: If you measure multiple teeth from the same individual, include tooth position to get medians per tooth. If you only want one median per individual, omit tooth position.

Choosing the right grouping columns ensures you get accurate and relevant results. If you're unsure, start specific — you can always simplify later.
